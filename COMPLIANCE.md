# Compliance & ethics

This engine is a **research and original-content tool**, not a re-upload tool.

## What it does (allowed)
- Reads **public** video metadata (titles, view/like/comment counts, descriptions)
  via the official **YouTube Data API v3**, under Google's API Terms.
- Uses that data as *inspiration* to generate **original** scripts and ideas you
  create and film yourself.

## What it deliberately does NOT do
- It does **not** download, scrape, copy, or re-upload anyone's video, audio,
  thumbnail, or script. Re-uploading others' content is copyright infringement
  and violates YouTube's Terms.
- It does **not** generate deceptive titles/thumbnails — the prompt explicitly
  forbids clickbait lies and fake claims.

## Your responsibilities
- Keep your own content original and truthful.
- Only mark videos `Published` after you've created and uploaded them yourself.
- Respect the YouTube Data API quota and Terms of Service.
- For any repurpose/clip workflow, only use footage **you own or are licensed**
  to use (your own uploads, Creative Commons with attribution, etc.).
